# Logistic Regression for Disease Prediction

## Objective
Predict disease outcomes using **Logistic Regression** and evaluate the effect of regularization.

---

## Methods Used
- Logistic Regression (L1 & L2)  
- ROC Curve & AUC  
- Cross Validation (for tuning)
