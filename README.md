## Overview
A hands-on collection of core Machine Learning implementations, covering the full workflow from **unsupervised learning → supervised learning → model evaluation**.

Focused on **understanding concepts through practical implementation**, not just using libraries.

| 03 | PCA | Dimensionality reduction |

| 07 | Random Forest | Ensemble learning |
