# Dimensionality Reduction using PCA

## Objective
Reduce high-dimensional image data using **PCA** while preserving maximum information.

---

## Methods Used
- Principal Component Analysis (PCA)  
- Explained Variance Analysis  
- Dimensionality Reduction

## Key Findings
- High-dimensional data projected into 2D for visualization  
- Dataset reduced to lower dimensions (e.g., 50D)  
- Minimum components selected to retain ~95% variance  

---

## Insight
PCA reduces dimensionality by keeping the most important variance, improving efficiency with minimal information loss.
